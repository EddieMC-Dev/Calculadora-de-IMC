# Calculadora-de-IMC
## Introdução 
Boa tarde pessoal, hoje trago mais um projeto de calculadora mas essa calcula o IMC (Índice de Massa Corporal) usado 
para classificar o peso do usuário em: abaixo do peso, peso normal, sobrepeso e obesidade.
